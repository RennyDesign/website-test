What is ColorMixer?

ColorMixer is a tool that helps you to solve the real world color mixing problems using its unique subtractive color model. It is able to simulate real-world color mixing and more importantly, it can “unmix” any given color into a combination of mixing colors.

Color Picker

Pick colors Color Wheel, color values in RGB, CMYK, or HSL model, an image from a gallery or camera, and your custom color table.

Unmix

Select a color from the color picker and the app will resolve it into a combination of prime colors.

Mix A + B

Select 2 colors and the app shows the result of mixing them together.

Convert A to B

The app gives you a list of color you need to add to convert one color into another.

The mix composition screen

Pick colors Color Wheel, color values in RGB, CMYK, or HSL model, an image from a gallery or camera, and your custom color table.
